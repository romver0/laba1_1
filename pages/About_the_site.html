Сайт сделан для выполнения заданий по дисциплине WEB-программирование ЧГУ 2 курс.

Автор: Зверев Роман Евгеньевич, группа ИВТ-41-20.

Основной задачей данного проекта являлась демонстрация:

  1. Умения работать с гипертекстовой раземткой HTML;
  2. Понимания принципов DOM и построения HTML документа;
  3. Умения создавать каскадные стилевые таблицы CSS, а именно:
    • Создание качественного и красивого офоромления;
    • Понимания правил CSS;
    • Подстраивание стилевых правил под разные устройства (оптимизация);
  4. Способности работать с Java Script, её библиотеками;
  5. Создания интерактива с помощью Java Script, дополнения возможностей не реализуемых стандартными средствами HTML и CSS.
  6. Вывод: ввиду уже имеющихся знаний по работе с вышеперечисленным, были закреплены уже полученные навыки, повторно пройден материал.
